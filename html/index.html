quick start

assuming you have the following (for example) strings for a list

char* s1="first";
char* s2="second";
char* s3="third";

you would create the list like this

clist_t *list=clistCreateList();
clistAddNode(list,s1);
clistAddNode(list,s2);
clistAddNode(list,s3);
